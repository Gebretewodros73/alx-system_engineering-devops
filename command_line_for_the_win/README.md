# CMD Challenge Repository

Welcome to the CMD Challenge repository! This repository contains a series of programming challenges practice on .jpg and .png file that are designed to test command-line interface (CLI) skills. Each challenge provides a specific task that needs to be completed using the command prompt or terminal in Windows, macOS, or Linux.

## About the Challenges

The CMD Challenge repository contains challenges of varying difficulty levels, ranging from beginner to advanced. Each challenge is designed to test different aspects of command-line skills, including file manipulation, network configuration, system administration, and more.

The challenges are intended to be fun and educational, providing a platform for users to practice and improve their CLI skills.
